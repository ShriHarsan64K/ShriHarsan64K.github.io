## ✨ Features

### 🎨 Visual Design

- **Realistic Milky Way backgrounds** in hero and contact sections
- **Rotating Earth with Saturn-style rings** (6,600+ particles)
- **45,000+ stars** with natural color variations
- **Twinkling starfield** animations
- **Custom GLSL shaders** for Earth day/night effect
- **Smooth animations** at 60 FPS

### 🛠️ Technical Features

- ✅ Fully responsive (mobile to 4K)
- ✅ Three.js WebGL 3D graphics
- ✅ AJAX contact form (no page reload)
- ✅ Beautiful gold success notifications
- ✅ Custom cursor effects
- ✅ GPU-accelerated animations
- ✅ Vanilla JavaScript (no frameworks)

## 🚀 How to Update Your GitHub Pages

### **Updates will reflect at:** https://shriharsan64k.github.io

### **Method 1: GitHub Website** (Easiest)

1. Go to: https://github.com/ShriHarsan64K/ShriHarsan64K.github.io
2. Click on `index.html` → Click ✏️ pencil icon
3. Delete all content, paste new version
4. Scroll down → Click **Commit changes**
5. Do the same for `README.md`
6. Wait 1-2 minutes → Hard refresh your site

### **Method 2: Git Command Line**

```bash
# Clone your repo
git clone https://github.com/ShriHarsan64K/ShriHarsan64K.github.io.git
cd ShriHarsan64K.github.io

# Replace files (copy new index.html, README.md here)

# Commit and push
git add .
git commit -m "Update to space-themed portfolio"
git push origin main

# Wait 1-2 minutes for GitHub Pages to rebuild
```

### **Check Deployment Status:**
https://github.com/ShriHarsan64K/ShriHarsan64K.github.io/actions

✅ Green checkmark = Deployed successfully!
